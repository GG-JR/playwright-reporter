# Changelog
All notable changes to this project will be documented in this file.

## 1.2.5

##### Added
- Added the option "consoleTestOutput". If this option is set to "true", the test result will be displayed on the console

##### Changed
- Added options to the video player to automatically start the video and run it in a loop
- Does not generate a report if no tests are found.

##### Removed
- Removed the "minifyAssets" option. Now all assets are minified.
- Removed the comma at the beginning of stdout and stderr.

##### [Download](https://www.npmjs.com/package/playwright-html/v/1.2.5)

## 1.2.0

##### Added
- The option to change the report theme

##### Changed
- The way you can open the test details (the preview button has been removed from the test list, now you have to click on the test line to open the details)

##### [Download](https://www.npmjs.com/package/playwright-html/v/1.2.0)
